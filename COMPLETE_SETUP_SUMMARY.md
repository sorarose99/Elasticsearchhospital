## ğŸš¨ Common Issues & Solutions

### Issue: "Cannot find module 'playwright'"
**Solution:**
```bash
npm run demo:install
```

### Issue: "Connection refused"
**Solution:**
```bash
# Make sure dev server is running
npm run dev
# Wait for "Local: http://localhost:3000/"
# Then run recorder in new terminal
```

### Issue: "Screenshots are blank"
**Solution:**
Edit `playwright-demo-recorder.js`, increase wait times:
```javascript
await page.waitForTimeout(5000); // Increase from 2000
```

### Issue: "Script runs too fast"
**Solution:**
Edit `playwright-demo-recorder.js` line 13:
```javascript
slowMo: 2000, // Increase from 1000
```

### Issue: "Video quality is poor"
**Solution:**
Close other applications, reduce browser window size, or record manually with OBS Studio.
